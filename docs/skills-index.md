# Skills Index

This is the **canonical public catalog** for the repository.

Each skill is listed exactly once and includes:

- `Primary Category`: its main place in the public taxonomy
- `Task Tags`: short workflow labels for fast scanning
- `Type`: `Curated` or `Custom`
- `Use for`: a compact description of when to reach for it

The category names below come directly from [taxonomy.md](taxonomy.md).

## Public Inventory Summary

- `34` curated upstream-derived skills
- `10` original custom skills
- `44` public skills total

## Outside the Public Catalog

These groups are intentionally **not** part of the current public inventory:

- proprietary format skills such as `pdf`, `docx`, `pptx`, `xlsx`
- unclear-license research search skills such as `openalex-database`, `paper-2-web`, `uspto-database`
- vendor-coupled search wrappers such as `research-lookup`, `parallel-web`, `perplexity-search`, `bgpt-paper-search`
- niche but not yet prioritized subfields such as `qiskit`, `cirq`, `pennylane`, `pymc`, `simpy`, `timesfm-forecasting`
