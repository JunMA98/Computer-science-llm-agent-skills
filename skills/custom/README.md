# Custom Skills

This folder contains original skills created specifically for CS research and programming workflows.

## Current custom skills

- `research-planning`
- `benchmark-design`
- `experiment-tracking`
- `code-reproduction`
- `reviewer-response`
- `agent-coding`
- `repo-cleanup`
- `github-release-prep`
- `paper-to-code`
- `code-to-paper`

## Why these skills were added

The curated upstream selection was already strong on libraries, writing support, and visualization, but still lacked several workflow-level abilities that are central to real research practice:

- planning and decomposition
- benchmark discipline
- run tracking
- reproduction
- rebuttal writing
- agent engineering
- repository cleanup
- public release preparation
- translation between papers and code
